# Institutional BI Governance Project

A self-directed analytics project simulating how a university Data & Analytics
team builds, governs, and maintains institutional dashboards. Built using
public IPEDS data to benchmark Oklahoma State University against peer public
land-grant institutions on enrollment, retention, and graduation outcomes.

## Live Dashboard

View the published dashboards on Tableau Public:
[Institutional Analytics: Enrollment & Student Outcomes](https://public.tableau.com/app/profile/hashika.venkatramanan7332/viz/InstitutionalAnalytics/InstitutionalAnalyticsEnrollmentStudentOutcomesO)

## Project Overview

This project was built to strengthen institutional research and Tableau
administration skills beyond day-to-day GRA work, using real public
higher-education data rather than synthetic or coursework datasets.

**Institutions analyzed:** Oklahoma State University, University of Arkansas,
Kansas State University, Iowa State University, University of Nebraska-Lincoln,
Texas Tech University, Mississippi State University, Auburn University,
University of Missouri.

**Time period:** 2018-2023 (6 academic years)

**Data source:** [IPEDS](https://nces.ed.gov/ipeds) (Integrated Postsecondary
Education Data System), U.S. Department of Education, National Center for
Education Statistics.

## Metrics Included

| Metric | Description |
|---|---|
| Total Enrollment | Total headcount enrollment per institution per year |
| Full-Time Enrollment | Students enrolled full-time |
| Part-Time Enrollment | Students enrolled part-time |
| Undergraduate Enrollment | Bachelor's-seeking undergraduate headcount |
| Full-Time Retention Rate | % of first-time, full-time students returning for year two |
| Graduation Rate (Total Cohort) | % of first-time, full-time students graduating within 150% of normal time (6 years for a bachelor's degree) |

## Dashboards

1. **Enrollment Trends** — Total enrollment trends across all nine
   institutions, 2018-2023, to visualize shifts in institutional size and
   enrollment composition over time.
2. **Retention & Graduation Rates** — Side-by-side comparison of student
   retention and completion outcomes across the same peer group.
